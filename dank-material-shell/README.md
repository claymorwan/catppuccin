# Catppucin for DankMaterialShell

Catppuccin color scheme for [DankMaterialShell](https://github.com/AvengeMedia/DankMaterialShell)

## Setup
1. Disable custom colors in settings
2. Enable the theme following [this guide](https://github.com/AvengeMedia/DankMaterialShell/blob/master/docs/CUSTOM_THEMES.md#setting-custom-theme)
